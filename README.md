## Adımlar

Klonla

```bash
  git clone https://github.com/kmaksut/typingtest.git
```

Klasöre git

```bash
  cd typingtest-main (klasör ismi değişkenlik gösterebilir)
```

Gerekli paketleri yükle

```bash
  python -m pip install Pillow
  python -m pip install ttkbootstrap
```

Çalıştır

```bash
  python app.py
```
